## WICHTIG: Was hat sich geändert?

### Kontext: Händler-Kampagne

Die Landing Page ist Teil einer **Händler-Kampagne**. Der Ablauf:

```
KORODUR → E-Mail an Händler (mit Aktionsangebot: 24 Sack geschenkt)
    → Händler schicken E-Mail an IHRE Kunden (mit Link zur Landing Page)
        → Endkunde landet auf dieser Seite
```

**Das Aktionsangebot (Frühjahrsaktion, 24 Sack geschenkt) gilt NUR für Händler, NICHT für Endkunden.**

### Was muss raus (vs. v1)?

| Element | v1 | v2 |
|---------|----|----|
| Frühjahrsaktion / Aktions-Badge | Überall prominent | **Komplett entfernen** |
| "24 Sack geschenkt" | Hero, Aktions-Sektion, FAQ | **Komplett entfernen** |
| "Nur bis Ende März 2026" | Mehrfach | **Komplett entfernen** |
| Aktions-Sektion (Sektion 7) | Eigene dunkle Sektion | **Komplett entfernen** |
| CTAs "Aktionsangebot sichern" | 8+ Stück | **Komplett entfernen** |
| Sticky Mobile CTA | "Aktionsangebot sichern" | **Entfernen oder anpassen** |

### Was bleibt / wird gestärkt?

| Element | Anpassung |
|---------|-----------|
| Problem-Darstellung | Beibehalten – Kernargument |
| Lösung / So funktioniert's | Beibehalten – Kernargument |
| Vergleichstabelle | Beibehalten – starkes Verkaufsargument |
| Referenzprojekte | Beibehalten – bester Social Proof |
| Zielgruppen-Tabs | Beibehalten – Relevanz für verschiedene Endkunden |
| Technische Daten | Beibehalten (kompakt) |
| FAQ | Anpassen: Aktions-FAQ raus, ggf. "Wo kann ich ARM kaufen?" rein |

## 1. Zielgruppe der Landing Page

**Endkunden**, die von ihrem Baustoffhändler eine E-Mail mit Link erhalten haben.

Typische Endkunden:
- **Kommunen / Bauhöfe** – reparieren Straßen, Gehwege
- **GaLa-Bau-Betriebe** – reparieren für ihre Auftraggeber
- **Facility Manager** – Parkplätze, Zufahrten, Hofflächen
- **Verarbeiter / Handwerker** – suchen zuverlässiges Material

**Einstiegssituation:** Der Endkunde hat gerade eine E-Mail von seinem Händler gelesen, in der ARM als Lösung für Frostschäden / Asphaltreparatur empfohlen wird. Er klickt auf den Link und will mehr erfahren.

**Erwartung:** Sachliche Produktinformation. Überzeugende Argumente. Belege (Referenzen, technische Daten).

---

## 2. Conversion-Ziel: Minimal und gezielt

**Die Seite ist KEINE Lead-Generierungs-Seite.** Der Endkunde soll beim Händler kaufen (der hat das Angebot).

### Einziger CTA (am Ende der Seite):

> **"Größeres Projekt? Wir beraten Sie gerne."**
> → Link zur KORODUR-Kontaktseite: https://www.korodur.de/kontakt/
> → Alternativ / zusätzlich: Link zu den deutschen Kontaktpartnern / Vertriebsregionen

**Platzierung:** Nur am Ende der Seite, NICHT im Hero, NICHT wiederholt.

**Sekundär (optional):**
- Datenblatt-Download (PDF) – bleibt als Micro-Conversion
- Link zu weiteren Referenzen auf korodur.de

### Was NICHT als CTA:
- Kein "Jetzt Aktionsangebot sichern"
- Kein Kontaktformular auf der Seite
- Kein "Jetzt beraten lassen" im Hero
- Kein Sticky Mobile CTA

---

## 3. Seitenstruktur (Empfehlung)

```
HEADER:    Logo KORODUR | Minimalste Navigation (Produkt, Referenzen, Kontakt)
HERO:      Problem + Lösung in einem Satz. Key Figures. Datenblatt-Download.
PROBLEM:   Warum Kaltmischgut nicht reicht (3 Karten)
LÖSUNG:    So funktioniert ARM (4 Schritte)
VERGLEICH: ARM vs. Kaltmischgut vs. Heißasphalt (Tabelle)
REFERENZEN: Bewährt in der Praxis (Projekte + Testimonial)
ZIELGRUPPEN: Für jede Fläche. Für jeden Einsatz. (Tabs)
TECHNIK:   Technische Daten (kompakt)
FAQ:       4-5 Fragen (ohne Aktions-FAQ)
CTA:       Größeres Projekt? Kontakt zu KORODUR (einmalig, dezent)
FOOTER:    KORODUR Kontaktdaten
```

### Hero – Neuer Fokus

- **Headline:** "Einmal reparieren. Dauerhaft halten." (ohne "Frostschäden?")
  - Oder: "Die dauerhafte Alternative zu Kaltmischgut."
- **Sub-Headline:** Rapid Set ASPHALT REPAIR MIX – schnell, einfach und dauerhaft. Begehbar nach 30 Minuten.
- **Key Figures:** 30 Min. begehbar | 30–600 mm | 38 N/mm² | 50+ Referenzen
- **Einziger Button:** "Datenblatt herunterladen (PDF)"
- **KEIN** Aktions-Badge, **KEIN** primärer CTA-Button

### FAQ – Angepasst

Raus:
- "Wie funktioniert die Frühjahrsaktion?" → ENTFERNEN

Neu/Anpassen:
- "Wo kann ich Rapid Set ARM kaufen?" → "Rapid Set ARM ist über den Baustoffhandel erhältlich. Fragen Sie Ihren Händler nach Verfügbarkeit und Konditionen."
- "Kann ich ARM selbst verarbeiten?" → Ja, einfache Verarbeitung, kein Fachpersonal nötig

## 4. Tonfall und Stil

- **Sachlich-informativ**, nicht werblich-dringlich
- **Keine künstliche Dringlichkeit** (kein "Nur bis...", kein "Jetzt sichern")

- **Händler-neutral** – die Seite nennt keinen bestimmten Händler
- Endkunde soll denken: "Das klingt überzeugend, ich frage meinen Händler"

## 7. Hinweise für die Agents

### Agent 1 (Stratege):
- Positionierung OHNE Aktionsangebot neu denken
- Storytelling: Problem → Lösung → Beweis (Referenzen)
- Endkunde kommt von Händler-E-Mail – hat bereits erstes Interesse
- Kein "hard sell", sondern "informed decision"

### Agent 2 (Content Creator):
- Hero-Text komplett neu (ohne Aktion)
- Alle CTAs entfernen bis auf einen dezenten am Ende
- FAQ anpassen (Aktions-FAQ raus, Kauf-FAQ rein)
- Ton: informativ statt aktivierend

### Agent 3 (SEO-Spezialist):
- Title/Description ohne "Frühjahrsaktion" und "24 Sack geschenkt"
- Schema Markup: kein Offers/Discount
- Keywords bleiben: Asphaltreparatur, Kaltasphalt Alternative, etc.

### Agent 4 (CRO-Optimierer):
- Seite ist NICHT auf Conversion optimiert – Conversion passiert beim Händler
- Nur ein CTA am Ende, kein Sticky CTA, kein Multi-Step-Form
- Fokus auf Informationsvermittlung und Vertrauensaufbau

### Agent 6 (Landing Page Builder):
- v1-Kurzversion (`index-short.html`) als Basis nehmen
- Aktions-Elemente entfernen, CTAs reduzieren
- Toggle-Pattern beibehalten (gute UX)
- Seitenstruktur straffen
